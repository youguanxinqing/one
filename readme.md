# one

一个脚本集合，每个脚本只关注一件小事儿。

# install

```bash
$ git clone https://github.com/youguanxinqing/one.git ~/.one
$ echo "export PATH=\$PATH:~/.one" >> ~/.bashrc
$ source ~/.bashrc
```

# usage

```bash
$ run++ main.cpp
cmd: g++ main.cpp -o main -std=c++14

hello world
```
